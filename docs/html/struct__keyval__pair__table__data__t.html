Detailed Description

First section in the table->table_data field, holds some misc. housekeeping data.

table->table_data layout/format:

The table->table_data field points to the buffer area passed to 'hashtable_create', and contains the following data:

+--------------------------—+ <— Lowest address of table->table_data | | | _keyval_pair_table_data_t | | | +--------------------------—+ | | | _keyval_pair_list_table_t | | | +--------------------------—+ | | | _keyval_pair_list_t table[] | | array items | | | +--------------------------—+ | | | _keyval_pair_data_block_t | | | +--------------------------—+ | | | data block data[] section | | | +--------------------------—+
